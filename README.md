# LOF 基金溢价监控工具 (LOF Premium Monitor)

一款轻量、纯前端的高颜值 LOF（上市开放式基金）溢价率实时监控工具。

## 🌟 核心特性

- **纯前端实现**：无需部署后端，直接用浏览器打开 `index.html` 即可运行（或配合本地简单 HTTP Server）。
- **多数据源 Fallback 策略**：
  - **场内行情**：直连东方财富 API，实时获取场内交易价格、成交额。
  - **场外净值**：优先尝试天天基金 JSONP 估算净值；对于无盘中估算的 QDII/商品型 LOF，自动 Fallback 至东方财富 Pingzhong 接口获取官方净值。彻底解决跨域报错和数据缺失问题。
- **防止 IP 封禁机制**：
  - 取消强制高频批量获取净值，改为纯手动触发。
  - 单独拉取时具备 400ms 请求间隔缓冲。
  - 使用内置 1 小时内存缓存 (`Memory Cache`)，支持行级单点强制刷新。
- **现代 Glassmorphism UI**：
  - 极简毛玻璃暗黑风（Dark Mode Glassmorphism），视觉清晰舒适。
  - 内置丰富交互状态：Toast 提示、行级 Loading、数字徽章（Badge）等。
- **灵活的数据交互**：
  - 精准支持按基金代码/名称搜索。
  - 支持快捷筛选（仅溢价、仅折价、显著偏差 ≥1%、净值未加载）。
  - 点击表头自由排序（支持溢价率、场内价、净值、涨跌幅、成交额）。
  - 内置辅助时间水印（精准显示场内价格时间和估算净值时间）。

## ⚙️ 核心逻辑公式

> **溢价率** = ( 场内最新价格 - 单位净值 ) / 单位净值 × 100%
> *注：溢价率为正代表场内买入成本高于当前净值（被高估）；为负代表折价（被低估）。*

## 🚀 快速上手

本项目无任何 NPM 依赖，下载即用。

1. **Clone 仓库：**
   ```bash
   git clone https://github.com/LinsVert/fund_lof.git
   cd fund_lof
   ```

2. **启动本地服务器（推荐）：**
   因为部分 JSONP 跨域请求在纯 `file://` 协议下可能受限于浏览器极强安全策略，建议起一个简单的本地 HTTP Server：
   ```bash
   # Python 3
   python3 -m http.server 8080
   
   # 或者 Node.js (需全局安装 serve)
   npx serve .
   ```
3. 在浏览器中打开：`http://localhost:8080`

## 📊 使用指引

- **刷新场内行情**：点击右上角「立即刷新」按钮，程序会获取东方财富目前所有 LOF 基金列表（按成交额排序，自动去重）。
- **获取净值**：
  - 如果只需要看特定的基金，可使用搜索框定位后，点击该行右侧的 **「查净值」** 按钮。
  - 如果想要全量更新，点击右上角的 **「💹 获取净值」** 按钮，工具会自动为你逐一点名获取，由于加入防封禁策略，约每秒2.5条，随时可点击取消。
- **高阶分析**：
  - 利用顶部四个仪表盘快速知晓全局情况（监控总量、溢价数量、折价数量、当前最高溢价）。
  - 使用右上角的下拉筛选直接过滤掉「净值未加载」的冗余项，或者找出「显著偏差 (≥1%)」获取套利/离场信号。
