# LOF 基金溢价监控工具 (LOF Premium Monitor)

> 🤖 **全权由 AI 构建**：本项目无论是纯前端的第一版，还是接入爬虫、定时任务、FastAPI和MySQL的现有多容器全栈架构，代码均**100% 由 AI (Google DeepMind Antigravity) 编写和重构完成**。

一款轻量、高颜值的 LOF（上市开放式基金）溢价率实时监控工具。帮助你在二级市场与一级市场之间寻找套利和折价机会。

## 🌟 核心架构与特性

- **Python 全栈重构**：从纯粹的前端静态页演进为 `FastAPI` (后端) + `MySQL 8` (数据库) + `Vanilla JS` (前端) 的全栈架构。
- **持久化与定时调度**：
  - 数据全部录入 MySQL，避免刷新即焚。
  - **APScheduler** 的后台定时任务加持，交易时段 (周一至周五 9-15点) **每 5 分钟自动更新一次场内大盘资金与价格**。
  - **每晚 21:00 全量更新一次净值**，省去人工逐个点击获取净值的苦恼。
- **多数据源 Fallback 机制**：
  - 后端 Python 使用 `httpx` 异步并发抓取东方财富和天天基金 API。
  - 由于部分 QDII/商品 LOF 缺乏盘中估值，遇到这部分基金会自动降级抓取历史官方净值。
  - 智能的容错和 JSON 解析防崩溃策略。
- **现代 Glassmorphism UI 前端**：
  - 极简毛玻璃暗黑风（Dark Mode Glassmorphism），视觉清晰舒适。
  - 内置丰富交互状态：Toast 提示、全量数据更新监控面板。

## ⚙️ 核心逻辑公式

> **溢价率** = ( 场内最新价格 - 单位净值 ) / 单位净值 × 100%
> *注：溢价率为正代表场内买入成本高于当前净值（被高估）；为负代表折价（被低估，存在套利或建仓机会）。*

## 🚀 极速部署 (Docker Compose)

本项目已完全容器化，你只需要安装好 Docker 即可在任何环境下一键跑起来：

1. **Clone 仓库：**
   ```bash
   git clone https://github.com/LinsVert/fund_lof.git
   cd fund_lof
   ```

2. **一键拉起后端、数据库、前端：**
   ```bash
   docker compose up -d
   ```
   *注意：初次启动 MySQL 8.4 可能需要数秒初始化。后端的连接池配置了最长 30 秒的探活动态重连安全锁，只需稍等片刻。*

3. **打开浏览器访问：**
   `http://localhost:8000`

## 📊 使用指引

- **前端状态同步**：前端页面每隔 15 秒会自动从后端拉取最新数据库的状态。
- **单独触发更新**：
  - **刷新行情**：点击左上角「立即刷新」，指示后端立刻并发去东方财富取全场 LOF 数据入库。
  - **单条净值**：通过搜索栏找到对应代码，点击行尾的 **「请求后端…」**，立刻针对此基金定向插队获取净值。
  - **全量同步**：点击顶部的 🚀 「全量更新净值(Debug)」强制触发每晚 9 点才跑的后台全量刷新，此时由于防封禁限速（每支基金隔0.5s），大名单可能需要几分钟才能全部拿到。
